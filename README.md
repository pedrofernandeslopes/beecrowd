# Soluções do BeeCrowd (URI Online Judge)

Este repositório contém soluções para os problemas do [BeeCrowd (anteriormente URI Online Judge)](https://www.urionlinejudge.com.br/). As soluções são fornecidas em quatro linguagens de programação: C, C++, Java e JavaScript.

## 🚀 Sobre o Projeto

O objetivo deste repositório é ajudar estudantes, professores e entusiastas da programação a entender e resolver os problemas propostos pelo BeeCrowd. Ao fornecer soluções em diferentes linguagens, espero tornar o material acessível a uma ampla gama de programadores.

## 🛠️ Linguagens Utilizadas

- **C**
- **C++**
- **Java**
- **JavaScript**

## 📁 Estrutura do Repositório

Cada problema tem sua própria pasta, nomeada com o ID do problema. Dentro de cada pasta, você encontrará soluções nas três linguagens mencionadas.

```
/problema-id
    /cpp
        solucao.cpp
    /java
        Solucao.java
    /javascript
        solucao.js
```
